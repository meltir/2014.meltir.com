Short blurb

My name is Lukas (AKA Meltir).

I'm a web developer, originally from Poland.

Short Bio

Born in Poland, Poznan in 1983.

After gaining some experience and some references, found a home at www.sote.pl where I stayed for 3 years, developing php e-commerce solutions and modifying existing web apps to better suit the needs of the company's clients.

Moved to London in 2008, where I joined Telecom plus PLC where I built up the webteam, crated a bunch of new properties, migrated some of the old ones to PHP, automated a bunch of stuff, implemented a bunch of API clients and applied a number of "best practices" over the course of 7 years.

Still in London and trying to do my own thing in Virtual Reality since 2015 !

A registered linux user (this used to be a thing...), fairly proficient in using older Debian releases, but lately found Ubuntu much more convenient.

Linux user by preference, but comfortable (if slightly cranky) using windows and xampp.

Playing around with nginx, AWS and GCloud when I'm bored.

Up until 2008 was a fluxbox evangelist and supporter, but for various reason made a switch to gnome and may not come back...

I love working on the backend, designing features and building api's that outlive the original spec.
